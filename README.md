# Feature Switch

<img src="./docs/logo.svg" width="560" />

Feature SwitchはWebブラウザの機能を無効化するChrome拡張機能である。

## インストール

[Feature Switch - Chrome ウェブストア](https://chromewebstore.google.com/detail/mgigamdejdjeldkimcjojpnaimfoahhi)

## ビルド

```bash
$ npm i
$ npm run zip
```
